# Card Games ŌĆö web UI

Svelte 5 + Vite app in the monorepo. Depends on `@card-games/scoundrel`.

- **Run:** from repo root, `npm run dev:web` (or `npm run dev` in this folder).
- **Shared card UI:** `src/lib/cards/` (start TDD helpers/components here).
- **Scoundrel screen:** `src/lib/scoundrel/` (`ScoundrelShell.svelte` is a thin controller smoke-test).
